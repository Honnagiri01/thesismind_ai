# Models Directory

This directory contains model files and configurations for ThesisMind AI.

## LLM Models

Supported large language models:

- Llama 2 (Recommended) - 7B, 13B, 70B parameters
- Mistral 7B - Fast and efficient
- Neural Chat - Conversation optimized
- Dolphin Mixtral - Multi-task capable

## Model Setup

### Using Ollama

1. Install Ollama from https://ollama.ai
2. Pull a model: `ollama pull llama2`
3. Start Ollama: `ollama serve`
